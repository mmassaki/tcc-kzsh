\chapter{\emph{Objective-C} e \emph{Cocoa Touch}} % (fold)
\label{cha:objective_c_e_classe_cocoa_touch}

\emph{Objetive-C} é a linguagem primária usada para construir \emph{softwares} para macOS e iOS baseada na linguagem C, com programação orientada a objeto. A origem dos conceitos são relacionados à linguagem \emph{Smalltalk}, uma das primeiras linguagens orientada a objeto.

\emph{Cocoa Touch} é um \emph{framework} de padrão de projeto utilizado no iOS com foco específico na interface orientada a toques. Fornece ferramentas básicas para implementar, como por exemplo o \emph{UIKit} que é o responsável pelo visual das aplicações do sistema operacional, como também gerenciamento de dados na aplicação, conexão, construção de \emph{strings}, acelerômetro, etc.

Para características mais específicas da aplicação é separada a parte em outros pacotes de desenvolvimento dentro do \emph{Cocoa Touch}. Alguns exemplos:
\begin{enumerate}
	\item \emph{Core Animation}
	\item \emph{Core Data}
	\item \emph{Core Audio}
	\item \emph{Core}
\end{enumerate}

Também tem como base o \emph{Core OS} e o \emph{Core Services}, utilizados no \emph{MacOS}.
