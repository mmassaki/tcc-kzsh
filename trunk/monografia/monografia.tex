\chapter{Desenvolvimento do projeto} % (fold)
\label{cha:desenvolvimento_do_projeto}

O projeto tem como objetivo o desenvolvimento de um sistema que permita o mapeamento do trânsito através do uso de dispositivo móvel com GPS. Para isso foram utilizados o dispositivo móvel \emph{iPhone} para que o usuário acesse o sistema e um servidor que gerencia e armazena os dados.

Para implementação do sistema foi utilizado a tecnologia \emph{Ruby on Rails} com banco de dados \emph{MySQL} devido a sua facilidade e agilidade de desenvolvimento de sistemas \emph{web}.

Como o sistema foi desenvolvido para a plataforma \emph{iPhone} é necessário que o usuário tenha disponível um aparelho destes para acesso ao sistema.
